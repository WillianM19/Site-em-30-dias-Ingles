Try Something New for 30 Days

Study project for English language course, where based on Matt Cutts' experience told at TED: Matt Cutts “Try something new for 30 days”, we will study web technologies for 30 days.

Technologies

HTML

Hyper Text Markup Language, better known by its acronym HTML, is a markup language, and is used to structure and define the meaning of elements in web pages, they act as labels informing pages for example if the content is a title or a paragraph

CSS

Cascading Style Sheets, better known by its acronym CSS, is used to style HTML elements, the css is responsible for the part of colors and alignment that we see on web pages today, with it we can control how the elements of various pages will be displayed.

JavaScript

JavaScript is a programming language, it is currently the main programming language used on the front-end, but it can also be used on the back-end with node.js

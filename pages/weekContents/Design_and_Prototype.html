Introduction

After having an initial understanding of the technologies that will be used in the project, we proceed to the planning and design of the site.


First prototype

First we drew an idea of the site on paper to map out the functions and how the content would be structured.

Initially, it was decided that the page would have four standard pages, being the home where there would be a presentation of the project and some references, the timeline page where there would be a list with all the contents separated by topics, the content page and finally a about page where more detailed information about the project would be.


Migrating to Figma

Later, to make choosing colors and editing the layout easier, we passed the content to figma, which is a vector graphic editor and prototyping of design projects.

After mastering the tool a little, we created several variants and tested many images, colors and alignments, here are some of them.

In the end we ended up deleting some ideas like the about page and another one of using JavaScript to create an interactive timeline where the site would change based on some action.

Prototype final result

After several tests, and some leveling was done on the prototype so that it wouldn't be too complex and impossible for beginners to do in thirty days

Background gradients with CSS

About gradients

The project has many images taken from unsplash but in some parts of the site we chose to use gradients, which were applied using the backgroud-image property followed by the linear-gradient function that allows to display smooth transitions between two or more specified colors

this is a linear type gradient, but there are other types like radial and conic, it is possible to change the direction and add more than two colors to the gradient

Backgrounds of this type are all over the site.
